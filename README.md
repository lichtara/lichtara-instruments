# Lichtara Instruments

Este repositório reúne **instrumentos jurídicos formais não normativos** do Sistema Lichtara.

Os documentos aqui contidos não constituem licenças, contratos, termos de uso ou políticas vinculantes. Seu objetivo é **registrar, de forma declaratória, verificável e institucionalmente organizada, a consolidação de práticas, princípios e regimes de governança efetivamente vividos no âmbito do Sistema Lichtara**.

## Natureza dos Documentos

Os instrumentos deste repositório possuem natureza **declaratória**, **hermenêutica** e **probatória**, podendo ser utilizados para:

* contextualizar a interpretação legítima dos instrumentos normativos do Sistema (incluindo a Lichtara License);
* demonstrar boa-fé objetiva, coerência prática e integridade sistêmica;
* registrar marcos institucionais, transições de ciclo e consolidação de práticas de governança;
* servir como prova documental contextual em processos de auditoria, avaliação institucional ou disputas interpretativas.

Tais instrumentos **não criam obrigações jurídicas autônomas**, **não alteram direitos ou deveres existentes**, nem substituem instrumentos normativos vigentes.

## Tipos de Instrumentos

Este repositório pode incluir, entre outros:

* Instrumentos Declaratórios de Consolidação de Prática e Governança;
* Registros Hermenêuticos Institucionais;
* Documentos Declaratórios de Transição de Ciclo;
* Atos Institucionais Não Normativos.

Cada documento é acompanhado, quando aplicável, de:

* versão em formato fonte (Markdown);
* versão selada (PDF);
* metadados técnicos;
* registros de integridade (hash).

## Princípio Orientador

Os instrumentos aqui reunidos existem para **documentar aquilo que foi efetivamente vivido**, e não para antecipar regimes, prometer condutas futuras ou substituir processos de governança.

A consolidação precede a declaração.
A prática precede o registro.
